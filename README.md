## Development

> [!IMPORTANT]
> 
> When using MyBatis Generator you'll need to replace:
> 
> ```kotlin
> import org.apache.ibatis.annotations.Mapper
> ```
> 
> with
> ```kotlin
> import org.mybatis.cdi.Mapper
> ```
